# 🎯 SIMPLE GUIDE - What You Need to Do

## 1️⃣ Run It Locally (5 min)

```bash
npm install
npm run dev
```

Open http://localhost:3000 ✅

---

## 2️⃣ Take Screenshots (20 min)

### Before (from original FOSSEE repo)

- Go to https://github.com/FOSSEE/workshop_booking
- Take 4 screenshots, save in `docs/screenshots/`

### After (from your running app)

- Desktop: 4 screenshots
- Mobile: 3 screenshots
- See `docs/SCREENSHOTS_GUIDE.md` for details

---

## 3️⃣ Deploy to Vercel (10 min)

```bash
npm install -g vercel
vercel
```

Or use https://vercel.com website

Copy your live URL ✅

---

## 4️⃣ Update README (10 min)

1. Add screenshots to README.md
2. Add your Vercel URL
3. Make sure all 4 questions are answered

---

## 5️⃣ Make Git Commits (20 min)

```bash
git add src/components/common/Button.*
git commit -m "feat(button): create Button component"

git add src/components/layout/
git commit -m "feat(layout): implement Header and Footer"

git add src/pages/
git commit -m "feat(pages): create all pages"

git add docs/screenshots/ README.md
git commit -m "docs: add screenshots and complete README"

git push origin main
```

Make 10+ commits showing progress ✅

---

## 6️⃣ Submit

You need:

1. ✅ GitHub repo URL
2. ✅ Live demo URL (Vercel)
3. ✅ Screenshots in README
4. ✅ All questions answered
5. ✅ Multiple git commits
