### How to Take Screenshots

#### Desktop Screenshots (1920x1080)

1. Open the page in Chrome
2. Press F12 to open DevTools
3. Press Ctrl+Shift+P (Cmd+Shift+P on Mac)
4. Type "Capture full size screenshot"
5. Save to this directory

#### Mobile Screenshots (375x667 - iPhone SE)

1. Open the page in Chrome
2. Press F12 to open DevTools
3. Click the device toolbar icon (or Ctrl+Shift+M)
4. Select "iPhone SE" from device dropdown
5. Press Ctrl+Shift+P (Cmd+Shift+P on Mac)
6. Type "Capture screenshot"
7. Save to this directory

### Screenshot Guidelines

- Use consistent device sizes
- Capture full page when possible
- Show interactive states (hover, focus) where relevant
- Include both light and dark mode if applicable
- Ensure no personal information is visible
- Use high quality (PNG format)
